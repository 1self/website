Request a sandbox API key

To integrate 1self, you need an API key. Please tell us your email, so that we can send you the key.


This API key will give you access to integrate with our Sandbox environment at http://sandbox.1self.co. It has the same integration features and interface as our live environment (https://api.1self.co. The key difference is that it’s not possible to create a 1self account in the Sandbox. We have done this so that developers can test the sorts of integrations and data to send to 1self without fear of corrupting a user’s live data.
