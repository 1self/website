IntelliJ IDEA

Download

Quickstart

1. Download the intellij plugin - it works with all IntelliJIdea versions

2. Open plugin preferences and choose install from disk

3. Select the plugin zip file and install it. Restart IntelliJIdea to load the plugin.

4. To see your IDE activity and build history charts, just click on 1self logo from 1self plugin toolbar(usually located at bottom right of IntelliJIdea window)
