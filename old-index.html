Get Started In Minutes

1. Create stream

The data stream is a write only event store with a unique url. When the user first opens the app, create a data stream. Store the url in your user profile.

2 Start logging

Log all your user events into the event stream. Data transfer is done through json objects. Choose event types from our library or invent your own.

3 Claim stream

Get the event stream url and show it to the user. The user can then claim the event stream by clicking on the link

javascript

var loggio = require('./loggio.js');
var stream = loggio.create('yourapp', 'yourappkey');
userProfile.url = stream.url;
stream.add({type: 'app:open', name:'yourapp'});
showUrlToUser(stream.url);
